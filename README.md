# Urban-Drive-Doc

## Contexte

### Dans une métropole saturée par le trafic, une des solutions est le VTC qui permet d’offrir une alternative de déplacements autre que la voiture ou les transports en commun.
### La solution d'Urban Drive se positionne comme le premier service de VTC entièrement fluide et simple. Chaque course compense ce problème de mobilité.
### L'application permet à l'utilisateur de réserver sa course afin qu’il puisse aller d’un point A à un point B. Cela permet aussi de créer de l’emploie local pour les chauffeurs VTC de notre plateforme.
